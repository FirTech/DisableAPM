# DisableAPM

一款轻量级的 Windows 工具，用于禁用机械硬盘的高级电源管理（APM），防止因频繁启停导致的延迟和噪音问题。

## 背景

你的机械硬盘是否在空闲一段时间后访问文件时发出"咔咔"的磁头归位声，然后盘片发出"叽～～"的加速旋转声？打开文件夹或加载游戏资源时是否会卡顿
1-2 秒？

这是因为现代硬盘默认启用了 APM（高级电源管理）。当硬盘空闲一段时间后，会自动降速甚至停转以节省电力。下次需要访问数据时，硬盘必须重新加速旋转，导致：

- 刺耳的咔咔声和旋转噪音
- 1-2 秒的卡顿/冻结
- 糟糕的用户体验

常见场景：

- 电脑空闲一段时间后打开机械硬盘上的文件夹
- 游戏将资源加载进内存后，几分钟后需要读取其他资源时突然卡顿
- 访问辅助存储盘上的文件

**DisableAPM** 通过禁用机械硬盘的 APM 功能来解决这个问题，让硬盘保持旋转状态，随时准备响应访问请求。

> **注意：** 程序会自动检测并跳过 SSD，因为 SSD 没有机械部件，不需要也不会受到 APM 设置的影响。

## 功能特点

- 自动检测并跳过 SSD 固态硬盘
- 可选择跳过 USB 外置硬盘
- 支持指定单个磁盘操作
- 资源占用极低
- 无需后台服务 - 每次启动运行一次即可

## 安装

### 下载预编译版本

从 [Releases](../../releases) 页面下载最新版本。

### 从源码编译

```bash
cargo build --release
```

编译后的二进制文件位于 `target/release/DisableAPM.exe`。

## 使用方法

### 以管理员身份运行

此程序需要管理员权限才能向硬盘发送 ATA 命令。

右键点击 `DisableAPM.exe` 并选择**"以管理员身份运行"**，或在提升权限的命令提示符中运行：

```cmd
DisableAPM.exe
```

### 命令行选项

```
DisableAPM.exe [选项]

选项:
    -i, --index <索引>    指定目标磁盘索引（从 0 开始）
    -u, --usb             包含 USB 外置硬盘
    -h, --help            显示帮助信息
    -V, --version         显示版本信息
```

### 使用示例

禁用所有机械硬盘的 APM（默认跳过 SSD 和 USB 硬盘）：

```cmd
DisableAPM.exe
```

禁用指定磁盘的 APM（例如磁盘 2）：

```cmd
DisableAPM.exe --index 2
```

禁用所有硬盘（包括 USB 外置机械硬盘）的 APM：

```cmd
DisableAPM.exe --usb
```

### 如何查看磁盘索引号

可以通过 Windows 磁盘管理查看磁盘索引：

1. 按 `Win + X` 键，选择"磁盘管理"
2. 查看每个磁盘左侧显示的编号（磁盘 0、磁盘 1、磁盘 2 等）
3. 使用显示的数字（从 0 开始计数）

或者使用 PowerShell 命令：

```powershell
Get-Disk | Select-Object Number, FriendlyName, BusType
```

## 工作原理

DisableAPM 直接向硬盘控制器发送 ATA `SET FEATURES` 命令（0xEF），子命令 0x85（禁用 APM）。这会指示硬盘保持旋转状态，随时准备立即响应访问请求，消除启动延迟。

使用的 ATA 命令：

- 命令寄存器：`0xEF` (SET FEATURES)
- 功能寄存器：`0x85` (Disable APM)
- 扇区计数寄存器：`0x00` (完全禁用)

## 兼容性

- **操作系统：** Windows 10/11 (x64)
- **硬盘类型：** SATA 机械硬盘，NVMe 固态硬盘会自动跳过
- **USB 硬盘：** 默认跳过（使用 `--usb` 参数可包含）
- **固态硬盘：** 自动检测并跳过

## 限制

- APM 设置不会持久保存，重启后需要重新运行此工具
- 部分 USB 硬盘盒可能不支持 ATA 透传命令
- RAID 卷可能不响应 APM 命令，具体取决于控制器

## 开机自动运行

要实现每次启动时自动禁用 APM：

1. 按 `Win + R`，输入 `shell:startup` 并回车
2. 创建 `DisableAPM.exe` 的快捷方式
3. 右键点击快捷方式 → 属性 → 快捷方式选项卡 → 高级
4. 勾选"用管理员身份运行"并点击确定

或者使用任务计划程序：

1. 打开任务计划程序（`taskschd.msc`）
2. 创建新任务，触发器选择"启动时"
3. 操作设置为运行 `DisableAPM.exe`，使用最高权限运行

## 安全性

- 仅影响机械硬盘
- SSD 会自动检测并跳过
- USB 硬盘默认跳过
- 工具仅发送 APM 禁用命令，不会修改任何数据
